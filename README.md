# Personal Portfolio (GitHub Pages)

This site hosts multiple resume variations for easy sharing.

## Publish on GitHub Pages

1. Create a new repo in GitHub (example: `hareendra5555.github.io` or `personal-portfolio`).
2. Copy the contents of this folder into the repo.
3. Commit and push.
4. In GitHub, go to **Settings → Pages** and set the source to `main` and `/ (root)`.
5. Your site will be available at:
   - `https://hareendra5555.github.io/` if the repo is named `hareendra5555.github.io`
   - `https://hareendra5555.github.io/<repo>/` for other repo names

## Customize
- Update contact links in `index.html`.
- Replace placeholder projects.
- Add/remove resume cards as needed.
